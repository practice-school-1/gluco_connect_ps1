# Gluco Connect PS1

## Team Workflow & Guidelines
* **No Direct Pushes:** Please do not push directly to the `main` branch. 
* **Branching:** Create a branch for your work (e.g., `feature/login`).
* **Monorepo:** Keep your work scoped to the `backend`, `patient-app`, or `doctor-portal` folders.
* **Pull Requests:** Submit a PR and require at least one review before merging.
